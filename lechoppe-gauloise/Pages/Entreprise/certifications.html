Questions Fréquentes

FAQ Certifications

Toutes nos certifications peuvent être vérifiées :

  • Numéros de certificat sur les sites des organismes accréditeurs
  • Scans haute résolution avec filigranes de sécurité
  • Contact direct avec les organismes certificateurs
  • Base de données publique accessible 24/7

Nous fournissons également des lettres de vérification sur demande.

Les durées de validité varient selon les certifications :

  • ISO 22000 : 3 ans (avec audits de surveillance annuels)
  • HACCP : Certification permanente avec audits annuels
  • HALAL : 1 an (renouvellement annuel)
  • Certifications nationales : 1 à 3 ans selon le pays

Toutes nos certifications sont maintenues à jour en continu.

Oui, nous proposons plusieurs options d'audit :

  • Audits virtuels : Visite guidée en vidéo avec partage d'écran
  • Audits sur site : Visite de nos installations (sur rendez-vous)
  • Audits documentaires : Revue complète de nos procédures
  • Audits avec tierce partie : Avec nos organismes certificateurs

Contactez-nous pour planifier un audit personnalisé.

Notre processus pour les nouvelles certifications :

  • Étude gratuite des exigences du pays cible
  • Proposition de certification adaptée
  • Gestion complète des démarches administratives
  • Délais d'obtention optimisés (généralement 4-8 semaines)
  • Pas de frais supplémentaires pour les clients existants

Nous avons déjà obtenu des certifications pour plus de 15 pays différents.
