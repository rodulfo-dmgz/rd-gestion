FAQ Export

Les délais varient selon la destination :

  • Union Européenne : 3-5 jours ouvrables
  • Amérique du Nord : 7-10 jours
  • Asie : 10-15 jours
  • Moyen-Orient : 8-12 jours

Tous les envois sont suivis en temps réel.

Nous fournissons toutes les certifications nécessaires :

  • Certificats d'origine
  • Certificats sanitaires
  • Documentation douanière complète
  • Certifications spécifiques pays (FDA, HALAL, etc.)
  • Factures commerciales et documents de transport

Oui, nous proposons plusieurs options :

  • Stockage en France : Entreposage réfrigéré jusqu'à expédition
  • Stockage local : Solutions dans certains pays via nos partenaires
  • Cross-docking : Transfert direct sans stockage intermédiaire
  • Gestion des stocks : Suivi et réapprovisionnement automatisés

Les minimums de commande varient selon la destination :

  • UE : 50 unités minimum
  • Hors UE : 100 unités minimum
  • Commandes personnalisables : à partir de 500 unités
  • Tarifs dégressifs selon volumes

Nous étudions également les petits projets sur demande.

  • Erreur de commande : Prise en charge complète
  • Dommages transport : Réclamation auprès de l'assurance
  • Retours commerciaux : Étude au cas par cas

Tous les retours sont gérés par notre équipe export.
